# HTMLPrettyPrinter

This is a simple HTML pretty printer that takes an HTML file and prints it in a more readable format. It is written in Bash.

## Usage

```bash
chmod +x script.sh
./script.sh <input_file>
```
### Example

```bash
./script.sh test.html
```
